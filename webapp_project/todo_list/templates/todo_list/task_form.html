Note: To make the form fields styled properly with Bootstrap, add a custom template filter in your Django app: 
    # In your app directory, create a file called templatetags/form_tags.py:
    
    from django import template
    register = template.Library()
    
    @register.filter(name='addclass')
    def addclass(field, css_class):
        return field.as_widget(attrs={"class": css_class})

Then load it at the top of your template with: {% load form_tags %}
